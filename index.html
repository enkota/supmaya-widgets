Supmaya Agent Popup

Lightweight popup widget that mirrors the flexibility of Tally's embed script while streaming your Supmaya agent directly inside an iframe.

Try it out

Click any control below to open the Supmaya popup.

Behavioral options

The popup can be opened automatically without any button clicks. Configure exactly when it should appear by setting the trigger options when initializing the widget.

  • Page load: Fire immediately when the DOM is ready.
  • Delay: Open after a specified number of seconds.
  • Exit intent: Detects when the pointer leaves the viewport.
  • Scroll position: Trigger once the user scrolls past a percentage of the page.

You can also control dimensions with width (default 376 px) and choose between a centered modal or a smaller docked popup in the bottom-right corner.
